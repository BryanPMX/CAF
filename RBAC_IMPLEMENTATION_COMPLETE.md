### **Enhanced Admin Handler** ‚úÖ

**File:** `api/handlers/admin.go`

**Before:**
```go

    c.JSON(http.StatusBadRequest, gin.H{"error": "Invalid role specified."})
```

**After:**
```go

    c.JSON(http.StatusBadRequest, gin.H{"error": err.Error()})
    return
}
```
